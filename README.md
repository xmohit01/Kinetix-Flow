# рҹҸӢпёҸвҖҚвҷӮпёҸ Kinetix-Flow

**Kinetix-Flow** is an AI-powered fitness tracking application. It transforms your device into a virtual personal trainer by using advanced computer vision to track your workout form, movement flow, and repetitions in real-time.

## вңЁ Features
* **Real-Time Pose Estimation:** Tracks body mechanics instantly using your camera.
* **Workout Flow Tracking:** Monitors the accuracy of your exercises.
* **Privacy-Focused:** All AI processing happens locally on your device; no video is uploaded to the cloud.

## рҹӣ пёҸ Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **AI / Computer Vision:** Google MediaPipe
* **Mobile Packaging:** Capacitor (by Ionic)
* **Build Tool:** Vite
